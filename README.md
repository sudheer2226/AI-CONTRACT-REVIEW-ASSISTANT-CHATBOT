# ⚖️ AI Contract Review Assistant Chatbot

> AI-powered legal contract analysis chatbot built using Legal-BERT and Gemini Flash for intelligent contract review and conversational assistance.

## 🎯 Problem Statement

Manual contract review is time-consuming and prone to human error.
This system automates contract analysis, clause extraction, and legal assistance using AI models.

## ⚡ Quick Demo

Input: Upload Legal Contract Document
Output: 📄 Extracted Clauses + ⚠️ Risk Analysis + 🤖 AI Legal Chat Assistance

## 🛠️ Tech Stack

* Python, Pandas, NumPy
* Legal-BERT (Contract Clause Extraction)
* Gemini Flash (AI Chatbot Assistance)
* Flask / Streamlit
* NLP & Transformer Models
* PDF/Text Processing Libraries

## 📊 Model Performance

| Metric         | Score                                         |

| Features       | Contract Analysis, Clause Extraction, AI Chat |
